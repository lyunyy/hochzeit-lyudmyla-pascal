# Hochzeitswebsite von Lyudmyla und Pascal

Mobile, zweisprachige Hochzeitswebsite für den 20. August 2026.

## Enthalten

- Deutsch und Ukrainisch mit Sprachwahl
- Live-Countdown bis zur Trauung
- Antragvideo mit erhaltenem Ton
- Direkte Routenlinks zu beiden Orten
- Vergrößerbare Fotogalerie
- Responsive Darstellung für Smartphone, Tablet und Desktop

## Lokal ansehen

Im Ordner `hochzeitswebsite` einen kleinen Webserver starten:

```bash
python3 -m http.server 8080
```

Danach im Browser `http://localhost:8080` öffnen.

Die Website besteht ausschließlich aus statischen Dateien und kann daher zum Beispiel über GitHub Pages, Netlify oder einen normalen Webspace veröffentlicht werden.
